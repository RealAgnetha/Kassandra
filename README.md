# Kassandra
Kassandra is a voice-controlled escape game for Amazon Echo Dot (Alexa). You do not need an Echo Dot to play, an Amazon account suffices. Although, an Echo Dot is highly recommended for an immersive experience. **Available only in German.**

Skill is live and available on the Amazon Skill Store: https://www.amazon.de/dp/B0FRFRC97G/


# Wie man spielt
Um das interaktive Escape Game **Kassandra** zu spielen, muss der Skill im Amazon Skill Store aktiviert werden: https://www.amazon.de/dp/B0FRFRC97G/

**Die Aktivierung und alle Materialien sind kostenlos.**

Bevor das Spiel gestartet wird, muss der Raum von einer Person, die selbst nicht mitspielt (denn dann kennt sie bereits die Lösungen) vorbereitet werden. Die Person druckt die Rätselhinweise aus und versteckt die Hinweise im Raum gemäß der Anleitung. Die Person(en), die dann spielen, bekommen die Spielregeln. Um Papier zu sparen, können die Regeln auch digital gezeigt werden, sie müssen nicht ausgedruckt werden. **Alle nötigen Dateien sind kostenlos in diesem Repo im Order *documents* verfügbar.**

Ohne diese Vorbereitung kann Kassandra nicht gespielt werden. 

Wenn der Raum präpariert wurde, kann **Kassandra** gestartet werden mit dem Befehl: *„Alexa, sag deiner Cousine, ich bin bereit!“*
